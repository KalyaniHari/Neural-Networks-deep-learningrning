# Neural Networks & Deep Learning

Coursework portfolio for a Neural Networks and Deep Learning course. It collects
the weekly in-class programming assignments (ICPs), a mini project, and exam
notebooks — all as Jupyter notebooks.

**Author:** Sri Kalyani Hari

## Contents

| Folder | Notebook | Description |
|--------|----------|-------------|
| `Week1/` | `NNDL_ICP1.ipynb` | ICP 1 |
| `Week2/` | `NNDL_ICP2.ipynb` | ICP 2 |
| `Week3/` | `NNDL_ICP3.ipynb` | ICP 3 |
| `Week4/` | `NNDL_ICP4.ipynb` | ICP 4 |
| `Week5/` | `ICP5.ipynb` | ICP 5 |
| `Week6/` | `ICP6.ipynb` | ICP 6 |
| `Mini_Project/` | `WhitePaperExperiments.ipynb` | Mini project — white paper experiments |
| `Exams/` | `Exam2_Code.ipynb` | Exam 2 code |

Each week folder contains its own `README.md` with assignment notes.

## Running the Notebooks

```bash
python -m venv .venv && source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install jupyter numpy pandas matplotlib scikit-learn tensorflow keras
jupyter notebook
```

Open any notebook from the folders above and run the cells top to bottom.
(Install additional packages as required by individual notebooks.)
